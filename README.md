# GitHub Actions Triggers - Daiana Fernández

## Descripción
Este repositorio contiene 6 workflows de GitHub Actions para demostrar distintos triggers.

## Workflows implementados

### 1. Push Workflow
Se ejecuta cuando se realiza un push a la rama main.
Mensaje:
Workflow activado por push

### 2. Pull Request Workflow
Se ejecuta al abrir un Pull Request hacia main.
Mensaje:
Pull request detectado

### 3. Issues Workflow
Se ejecuta cuando se crea un issue.
Mensaje:
Nuevo issue creado

### 4. Issue Comment Workflow
Se ejecuta cuando se comenta un Pull Request.
Mensaje:
Comentario en PR detectado

### 5. Workflow Dispatch
Se ejecuta manualmente desde GitHub Actions.
Incluye selección de nivel de alerta.

### 6. Schedule Workflow
Se ejecuta de manera programada.
Mensaje:
Ejecucion programada

Se adjuntan capturas de ejecución de cada workflow.
